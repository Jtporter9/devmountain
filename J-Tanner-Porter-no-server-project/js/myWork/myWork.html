  • Wordwire.me

    This is real time chat app that I created using the MEAN stack. I built this app in two weeks as part of a personal project while in attendance at Dev Mountain. In this app I used many a lot of fun things that help me learn a lot about the MEAN (Mongodb, Express, Angular, Node) stack, and in particular about using models in the back end and how to reference to each other. I used 3 different very important and key models in this project. Number one was 'Users', two was 'Conversations', and three was 'Messages', building the models and relationships for these was an intersting task. I also used Socket.io to help the app become a real time messaging app. If I would have had more time I would have liked to implement, file upload via Amazon S3, and a better notification system to notify you when you get a message. I also have plans to make this into a mobile app using Ionic.

  • Task Manager

    This is a simple but very functional task editor that I made in the begining of my time at Devmountain. It uses AngularJS, with your basic Html, and Css. This app allows you to create, edit, delete, and manage cards for a more organized daily effort. You can move cards from To do, to Doing, then into the done section. This project shows my comprehension of simple javascript functions and how to manipulate data using objects, and arrays. This was a fun project build on a saturday in october in 2015.
